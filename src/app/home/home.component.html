Welcome to Log Admin

This project contains two applications. A Spring Boot REST API with a postgreSQL database and a Angular front-end.

Install

Clone the spring boot repositort and also the angular front-end.

Dependencies

- Angular (9.1.0)

- Node (v12.16.1)

- Jdk (13)

- PostgreSQL

Run

To run the API: Open the project in your prefered IDE, let maven install all dependencies and run. The JPA will automatically generate the database following the Log model. Don't forget to change your postgree password and user in application.properties.

To start the front-end: Run ng serve for a dev server. Navigate to http://localhost:4200/. The app will automatically reload if you change any of the source files.
